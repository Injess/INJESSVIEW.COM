## 🚀 How to Use

### Accessing the Admin Panel
1. Navigate to: `your-domain.com/admin-login.php`
2. Login with existing credentials from the `users` table
3. Access the dashboard and all management features

### Quick Actions
- **Edit Pages**: Click "Manage Pages" → Select page → Edit content → Save
- **Upload Media**: Click "Media Library" → Choose file → Upload
- **Change Settings**: Click "Settings" → Update fields → Save
- **Create Users**: Click "Users" → Enter username/password → Create
- **Backup Site**: Click "Backup & Deployment" → Create New Backup

### Deployment Process
1. Create a backup before making changes
2. Test changes locally
3. Upload via FTP or Git
4. Verify on live site
